01 — Capability Surface

Что фактически даёт вам этот стек

Не «ещё один блокчейн». Это вертикально интегрированная финансовая платформа: USDC как нативная валюта, инфраструктура для кошельков и расчётов, и LLM-friendly документация. Главные строительные блоки:

L1 · Arc Network

USDC = gas

EVM-совместимая цепь (Reth + Malachite Tendermint). Sub-second finality, предсказуемые комиссии в USDC, нет волатильности ETH. Testnet, есть faucet.

Cross-chain

CCTP V2 + Gateway

CCTP burn-and-mint native USDC между 19+ цепями (Fast ~8-20s). Gateway даёт unified balance с <500ms transfers и 7-day trustless withdrawal — никакой кастодии.

Agentic payments

Nanopayments × x402

Sub-cent платежи (от $0.000001), gas-free, EIP-3009 offchain authorizations, batched settlement. HTTP 402 protocol — буквально просто middleware на Express.

Wallets

Passkey + Web2 login

Modular Wallets (ERC-4337/6900, passkey signer, batch + parallel ops), user-controlled с Google/Apple/email/PIN, или developer-controlled для автоматизации.

Gas abstraction

Gas Station + Paymaster

Спонсорируете gas с карты или даёте платить gas в USDC через ERC-4337 paymaster. Без нативного токена для пользователя.

Contracts

Templates + Events API

Audited ERC-20/721/1155 templates, deploy через REST/SDK, ABI calls без Solidity. Event Monitoring webhooks для реактивной логики.

Merchant flow

Managed Payments

Continuous payment intents для приёма USDC, stable deposit addresses per merchant, webhooks для подтверждений. Аналог Stripe Checkout, но для стейблов.

AI Co-pilot

MCP + Skills

Circle MCP сервер (api.circle.com/v1/codegen/mcp) даёт IDE/агенту живые signatures и адреса. 9 LLM-оптимизированных skills с архитектурными гайдами.

02 — Ten Ideas to Vibe-code

Что я бы собирал

Каждая идея привязана к конкретным primitives и Skills. Указан реалистичный MVP-скоуп и сложность. Топ-3 (MicroPay AI Gateway, Stripe-for-Agents, v0-for-Solidity) — мои фавориты: яркая ниша, защитимый USP, малое окно конкуренции.

01

MicroPay AI Gateway

Stripe Connect для агентских API. Per-token, per-call, sub-cent.
Прокси-слой между AI-агентом и любым LLM/data/compute API. Сервер выдаёт 402 Payment Required, агент подписывает EIP-3009, проксирует запрос. Owner маркетплейса собирает комиссию. Никаких подписок, никаких API-ключей — pay-as-you-go on the wire.
MVP scopeExpress middleware для x402 sell-side + buy-side SDK для агента. Один live endpoint, например, прокси GPT-4 с биллингом $0.00003 / 1k tokens. Деплой за выходные.

02

Stripe-for-Agents

Бюджеты, allowlists и аудит для AI-агентов с собственным кошельком.
Dashboard для команды, у которой бегают N агентов: каждому — отдельный passkey-wallet с allowlist-модулем (только разрешённые контракты/адреса), дневной лимит, real-time лог транзакций. Решает реальную проблему: «как дать GPT доступ к деньгам и не разориться».
MVP scopeWeb-консоль (Next.js + Modular Wallets SDK), создание агентского кошелька с allowlist-модулем, dashboard с лимитами, фид через Event Monitoring webhooks. Compliance Engine для скрининга outbound transactions.

03

v0-for-Solidity

Natural-language → задеплоенный DApp на Arc за 30 секунд.
Chat-интерфейс типа v0.dev, но генерирует и деплоит смарт-контракты. «Loyalty с 5% cashback в USDC» → MCP-сервер возвращает код по template, Circle Contracts API деплоит на Arc, dev-controlled wallet платит gas. Юзер получает адрес контракта и UI для админки.
MVP scopeNext.js chat UI, оборачивает Circle MCP server, использует templates (ERC-20/721/1155 + Airdrop), деплой через Smart Contract Platform API, авто-генерация admin-панели для read/write функций.

04

PayLink Universal

Stripe Payment Links для USDC. Любая цепь → ваш Arc-аккаунт.
Создаёте ссылку pay.example/r/abc?amt=42&memo=.... Плательщик открывает, App Kit определяет его кошелёк/цепь, через Gateway или CCTP отправляет USDC, settle на Arc-аккаунт мерчанта. На стороне мерчанта — webhook с payment intent ID. Чек, QR, recurring — всё стандартное.
MVP scopeNext.js: создание ссылки с amount/memo, страница checkout с App Kit виджетом, Managed Payments continuous intent на сервере, webhook handler, dashboard с историей.

05

Streampay

Зарплата по секундам. Sablier, но для USDC и без gas-боли.
Контракт-стример на Arc: startStream(recipient, ratePerSecond, end), получатель в любой момент клеймит накопленное. Так как Arc даёт sub-second finality и USDC как gas, payroll за реальное отработанное время становится дешевле обычного банковского перевода.
MVP scopeSolidity-контракт на Arc Testnet (Foundry), Next.js UI для создания стримов с modular wallet, dashboard для получателей с claim-кнопкой, Paymaster чтоб gas платился из самого стрима в USDC.

06

Drop-in Wallet Button

«Pay with USDC» виджет для любого SaaS в одну строку.
Один <script> тег → на сайте появляется кнопка. Пользователь логинится Google/Apple, получает modular wallet с passkey, может оплачивать USDC. Gas Station покрывает gas (или USDC paymaster), мерчант получает payment intent. Embeddable как Stripe Elements.
MVP scopeVanilla JS-виджет (~30KB), бэкенд на Cloudflare Workers для intent-ов, integration docs. Демо-сайт с «купить кофе за $4 USDC». Используем Web SDK для Modular Wallets.

07

TipStream

Стримерские тиры по секундам просмотра — gas-free, sub-cent.
Зритель депонит $5 USDC, включает «tip-mode» на стриме за $0.001/sec, пока смотрит. Подписи offchain через x402, settlement раз в N минут через Gateway. Стример видит pool в реальном времени. Идеальный fit для Twitch/YouTube/Substack-альтернатив, где CC-rails не работают на $0.05 чаевые.
MVP scopeБраузерный плеер на React, виджет «start tipping», OBS-overlay с pool counter. Sell-side x402 middleware, agent-style buy-side в браузере. Демо со стримером на тестовых USDC.

08

Onchain Cashback Engine

Loyalty-программа за час: ERC-1155 reward + event-driven кэшбэк.
Мерчант подключает свой USDC payment intent, эмитит ERC-1155 reward токены. Любая покупка → Event Monitoring webhook → дев-контролируемый кошелёк мерчанта минтит N токенов клиенту. Клиент в любой момент redeem'ит за USDC через Gateway, тратит на любой цепи.
MVP scopeDeploy ERC-1155 reward template на Arc, backend на Node.js слушает webhook'и Managed Payments → автоматически минтит токены. Merchant-консоль (Next.js) с настройкой rate (X% cashback).

09

Agent Job Escrow

Upwork для AI-агентов с on-chain escrow и milestone-платежами.
Заказчик постит задачу с бюджетом, депонирует USDC в escrow-контракт на Arc. AI-агент (или человек) берёт работу, на промежуточных milestone'ах получает stream через x402. По завершению — release или dispute (с view-key аудитом через будущий privacy module).
MVP scopeSolidity escrow на Arc с функциями deposit/release/dispute, dev-controlled кошелёк как арбитр (сначала ручной, позже AI-судья), Next.js marketplace UI. x402 для milestone-стримов агентам.

10

Multi-Currency Invoice

B2B-инвойс в EUR, оплата USDC, settlement в EURC. Без посредников.
Поставщик выставляет счёт в EUR, плательщик платит USDC с любой цепи, Arc Stablecoin Services делает FX в EURC и settle на EURC-баланс получателя. Решает реальную боль freelance / cross-border B2B: SWIFT + WISE + конвертация × 3-7 дней → 5 секунд.
MVP scopeNext.js + Managed Payments с continuous intent в EUR, App Kit для swap USDC→EURC на Arc, экспорт PDF-инвойса. Тут critical: Arc сейчас testnet, так что MVP на тестовых EURC/USDC — но архитектурно production-ready.

Стартовать сегодня

Самый короткий путь — поставить Circle MCP в Cursor/Claude Code и Circle Skills, потом начать с любой идеи выше. MCP сразу даёт live SDK-сигнатуры и адреса контрактов, skills дают архитектурные паттерны.

Arc Testnet и Sepolia (USDC) бесплатные — можно прокатить полную flow без вложений. EURC и Solana USDC тоже на тестнете.

# Установить Circle Skills (Claude Code)
/plugin marketplace add circlefin/skills
/plugin install circle-skills@circle

# MCP в Claude Code или Cursor
claude mcp add --transport http circle \
  https://api.circle.com/v1/codegen/mcp \
  --scope user
